# Phone Book #
A collaborative project for wyncode Rails breakouts

### Ruby Version ###
- Uses `2.5.0` Change version in `Gemfile` if needed

## Getting Stated ##
### Install dependencies: ###
- Run
```bash
bundle install
```
### Building Database ###
- Run
```bash
bundle exec rake db:setup
```
or
```bash
bundle exec rails db:setup
```
### Running App ###
- Run
```bash
bundle exec rails s
```
